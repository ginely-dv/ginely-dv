# Proyecto de Auditoría Interna de Ciberseguridad

Este proyecto forma parte de mi curso de ciberseguridad. El objetivo es recrear un proceso de auditoría interna utilizando datos ficticios proporcionados en el curso.  

Incluye:
- Un reporte formal de auditoría (PDF y versión Markdown)
- Datos utilizados en el análisis
- Apéndices y documentación adicional
- Evidencias del proceso

Este repositorio está diseñado para mostrar mis habilidades de documentación, análisis y uso profesional de GitHub.
